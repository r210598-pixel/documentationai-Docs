**Query Loop** is a WordPress block that dynamically displays a list of posts based on specific query rules.

Here’s how it works:

- The **Query Loop** defines *what* content to fetch (post type, categories, tags, order, number of posts, etc.).

- Inside it, the **Post Template** acts as a blueprint for *how each post is displayed*.

- The layout blocks inside the Post Template (like **Row Layout**, **Section**, columns, images, titles, meta, buttons) are repeated automatically for **every post returned by the query**.

- You design the structure **once**, and WordPress renders it multiple times—one time per post.

### **Count up**

Count Up Block (Kadence) is a dynamic content block used to display animated numerical values that count up from a defined starting number to a target value when the block becomes visible.

<Image src="https://blob-cdn.documentation.ai/org-5841073b-1314-4cc5-b42c-8bb574c4af65/doc-d75f3731-d9f4-45c6-a365-aecc769fe80b/1773410691352-9yzzy44hysi-30.png?q=85&fm=auto&auto=compress%2Cformat" width="1598" height="667" alt="" />

The block allows configuration of:

- starting and ending numbers

- number prefixes and suffixes (e.g. %, €, M€, +)

- animation duration and easing

- number formatting (such as thousand separators)

It is typically used to highlight key metrics, statistics, or performance indicators (e.g. revenue, percentages, headcount). The animation runs on page load or on scroll into view, helping draw user attention while keeping the content fully editable and lightweight within the WordPress block editor.

The cision-preview\.php file is a template part responsible for rendering a single Cision post preview item. It is used to display press releases or news entries within post listings, archives, or Gutenberg blocks.

What does the code do:\
Gets the terms (taxonomies) of the current record\
$categories = get\_the\_terms(get\_the\_ID(), 'cision\_category');

$tags       = get\_the\_terms(get\_the\_ID(), 'cision\_tag');

This means that for the current post, the Loop gets:

1. categories from cision\_category

2. tags from cision\_tag

Displays an HTML card \<article class="cision-item">

The markup uses Foundation classes (grid-x, cell, small-12, large-3/9, grid-padding-x) — this is a two-column layout on desktop and a single column on mobile.

Left column (preview image)

Post link (the\_permalink()), title is retrieved via the\_title\_attribute()

A custom function is called inside:

get\_attachment\_fallback(get\_post\_thumbnail\_id(), 'medium', \['class'=>'cision-item\_\_thumb'])
