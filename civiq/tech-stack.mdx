## Overview

CiviQ is built with a modern JavaScript web stack composed of a React frontend, a Node.js backend, and a document database.

## Frontend

- React
- Tailwind CSS
- React Router

## Backend

- Node.js
- Express.js

## Database

- MongoDB (Mongoose)

## Other libraries and services

- JWT
- bcrypt
- Cloudinary
- Geo API
